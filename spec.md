# SENSEI.LOG 仕様書（Ver. 4.0）

> 最終更新: 2026-05-16

## サービス・コンセプト

「静寂と内省の黒の秘書 ―― 自己を複製する羅針盤」

- **実用的な調和:** 空白の時間を「リソース（タスク消化時間）」と定義し、淡々と「公」と「私」を整理する
- **脳への定着:** 付箋を貼り替えるような手動操作により、一日の流れを主体的に脳内へロードする
- **自己の結晶化:** 日々のログから「思考の癖」を抽出し、最終的に自分自身の考え方をコピーしたAI（分身）を生成する

## データ設計（localStorage）

| キー | 内容 |
|------|------|
| `sensei_user_events` | ユーザー追加イベント（JSON配列） |
| `sensei_reflections` | 振り返りログ（JSON配列） |
| `sensei_day_patterns` | 日付ごとの時間割パターン（JSONオブジェクト） |
| `sensei_schema_v2` | スキーマバージョン（初回リセット制御） |
| `sensei_habits` | 習慣チェックデータ（日付→習慣ID→評価値） |
| `sensei_greeting_shown` | 今日の挨拶を表示済みか（YYYY-MM-DD） |
| `sensei_preset_*` | 初期プリセットデータの投入済みフラグ |

振り返りログは「良かったこと」のみに特化。AI学習データのノイズを最小化し、自己複製の精度向上を目指す。

## UX設計原則

- 自動調整に頼らず「手で貼り替える」ことで記憶への定着を図る
- 空き時間は「余白（リソース）」として美しく・ドライに表示
- 夜のリフレクション入力はスマートフォンで完結できること
- タッチターゲットは44px以上（Apple HIG 準拠）
- iOSでのフォーム入力時に自動ズームが起きないよう font-size:16px 以上
- 夜間使用を想定した低刺激デザイン（高輝度色を避ける）
- 昼間の屋外での視認性（文字コントラスト・フォントサイズ）を確保

## 習慣チェックシステム

### 評価タイプ

| タイプ | 対象項目 | 選択肢 |
|--------|---------|--------|
| `binary` | 明確にした/しなかった判断ができる項目（18項目） | できた ✓ / できなかった ✗ |
| `scale` | 程度・質が重要な項目（6項目） | 100% / 80% / 50% / 20% / ✕ |

### スコア計算

- **binary**: できた=100点、できなかった=0点
- **scale**: 100%=100点、80%=80点、50%=50点、20%=20点、✕=0点
- **平均スコア**: 回答済み項目のみを対象に平均値を算出（未回答は除外）

### カテゴリ別項目一覧

**🌙 睡眠・生活リズム（binary×4）**
- 起床・就寝時刻を守れた
- 6時間以上の睡眠が取れた
- 湯船に入った
- 就寝2時間前までに食事を終えた

**🥗 食事・ヘルスケア（scale×1、binary×4）**
- 糖質・砂糖の摂取を控えた（scale）
- 揚げ物を食べなかった
- 体重を測って記録した
- デンタルフロスをした
- 保湿・日焼け止めをした

**💪 運動・姿勢（binary×2、scale×1）**
- 20分以上の軽い運動をした
- 筋トレをした
- 仕事中に良い姿勢を保てた（scale）

**🧹 習慣管理（scale×1、binary×3）**
- 部屋が整理整頓・清潔な状態だった（scale）
- ゴシップ・ショート動画を見なかった
- 中毒性課金をしなかった
- 承認欲求目的のSNS投稿をしなかった

**📝 仕事術・対人（scale×3、binary×5）**
- 2分以内の用事をすぐにやれた（scale）
- 朝活を30分以上した
- Good & Newを書いた（または話した）
- 身だしなみを完璧に整えた
- 身近な人の話を集中して聴けた（scale）
- 人の良いところを見つけて伝えた
- 笑顔で元気にあいさつした
- 余白のある伝え方ができた（scale）
