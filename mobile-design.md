# SENSEI.LOG スマホ画面設計

> 作成: 2026-05-10 ／ 最終更新: 2026-05-16（iPhone 15 実機フィードバック反映）

---

## 画面構成の全体像

スマホは **4タブ構成**（右サイド浮遊ピル）。サイドバー・右パネルはすべて非表示。

```
┌─────────────────────────┐
│  ヘッダー (44px)        │  SENSEI.LOG ／ 日付 ／ ＋ボタン（円形）
├──────────────────────┬──┤
│                      │ 🗓 │  ← 右縦ピル（カレンダー）
│   コンテンツ         │ 📝 │  ← 右縦ピル（ログ）
│   (スクロール可)     │ ✓  │  ← 右縦ピル（チェック）
│                      │ 月  │  ← 右縦ピル（月）
└──────────────────────┴──┘
```

**起動フロー（1日1回・モバイルのみ）:**
1. 今日の一言オーバーレイ（哲学メッセージ + 今日の問い + 夕方の振り返り点）
2. 習慣チェックオーバーレイ（24項目・強制）→「今日を始める」or「スキップ」
3. カレンダー本体へ

## タブ 1：カレンダー（日カレンダービュー）

### 週間ストリップ（カレンダータブ時に上部に表示）

```
┌──────────────────────────────────────────┐
│  日 月 火 水 木 金 土  ← 横スクロール対応  │
│  11 12 13 14 15 16 17                    │
│  ● ●        ●                           │  ← イベントのカテゴリ色ドット
└──────────────────────────────────────────┘
```

- アクティブな日はcyan背景で強調
- 今日はアンダーライン
- タップで選択日を切替 → `selectedDay`・`mobSelectedDay` を同期

### カレンダー本体

```
┌──── gutter ────┬─── day-col ───────────┐
│  時間軸        │  イベント列            │  ← 縦スクロール
│  0:00〜24:00   │  (borderLeft 1px)    │
│                │                       │
│  08:00 ────────│───────────────────────│
│  09:00 ────────│  [朝学] 08:30         │
│  ...           │  [1限 現代文B]        │
│                │  [部活 吹奏楽]        │
│  現在時刻 ─────│──●──                 │  ← 赤点＋横線
└────────────────┴───────────────────────┘
```

### 時間軸設定

| 項目 | 値 |
|------|-----|
| 表示開始 | 00:00 |
| 表示終了 | 24:00 |
| 1分あたり | 1.5 px |
| 起動時スクロール位置 | 現在時刻の少し上 |

### イベントブロック（モバイル最適化）

```
[タイトル          HH:MM-HH:MM]
← flexレイアウトで横幅を使い切る
```

```css
display: flex; justify-content: space-between; align-items: baseline;
width: 100%;
```

- タイトル: カテゴリ色・bold・overflow ellipsis
- 時刻: ミュート色・monospace・右寄せ

```
├─────────────────────────────────────┤

├─────────────────────────────────────┤
│  🌙 睡眠・生活リズム                │  ← カテゴリラベル
│                                     │
│  起床・就寝時刻を守れた             │  ← binary項目
│  [できた ✓]  [できなかった ✗]      │

│                                     │
│  糖質・砂糖の摂取を控えた           │  ← scale項目
│  [100%][80%][50%][20%][✕]          │

│                                     │
│  ...（24項目続く）                  │
└─────────────────────────────────────┘
```

### binary型ボタン

```css
.hb-btn { flex:1; height:44px; border-radius:9px; font-size:13px; font-weight:700; }
.hb-yes   { color: rgba(74,222,128,0.7); }  /* 非選択 */
.hb-yes.sel { background: rgba(74,222,128,0.22); color: #4ade80; }  /* 選択済 */
.hb-no    { color: rgba(248,113,113,0.7); }
.hb-no.sel  { background: rgba(248,113,113,0.2); color: #f87171; }
```

### scale型ボタン（5段階）

```
100% → green  /  80% → cyan  /  50% → amber  /  20% → red(dim)  /  ✕ → red(dimmer)
```

## UX 原則（スマホ固有）

- タッチターゲット **44px 以上**（Apple HIG 準拠）
- 入力フォームの `font-size: 16px` を厳守（iOS の自動ズーム防止）
- サイドバー・右パネルは **完全非表示**（CSS で `display:none`）
- ナビゲーションは**日単位**（PC週表示と異なる）
- スワイプジェスチャーで前後の日に移動（touchstart/touchend で判定）
- **右手親指優先**: ナビゲーションピルを右サイドに配置
- **夜間使用前提** — 高輝度色は避け、宇宙紺ベースで目への刺激を最小化
- **昼間屋外対応** — モバイルでテキストコントラストを強化（`--text-muted` を `#8898b0` に）
- 振り返り入力は**ログタブ1画面で完結**（今日・昨日の2件同時表示）
